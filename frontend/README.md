# pronostidamus frontend

Frontend base del MVP construido con React, Vite, TypeScript y Yarn.

## Scripts

- `yarn dev`: inicia el servidor de desarrollo.
- `yarn build`: compila el proyecto para produccion.
- `yarn lint`: ejecuta ESLint.
- `yarn preview`: sirve el build generado localmente.

## Variable de entorno

Crear un archivo `.env` en `frontend/` con:

```env
VITE_API_URL=http://localhost:3000
```

Tambien puedes partir de `.env.example`.
